For my check password algorithm I created a CheckPassword class that has two functions. The first function is the check password function which takes in 5 parameters. The password as a string, the minimum length of the password, the minimum number of letters and numbers the password must contain, and the minimum number of secial characters the password must contain. The second function is a static function which returns nothing but prints text onto the console soliciting strings from the user.
